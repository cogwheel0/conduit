Conduit è un client mobile nativo open source per Open-WebUI. Connettiti al tuo server per chattare con i modelli AI, gestire le conversazioni e portare la tua AI self-hosted sempre con te, in modo sicuro e ovunque tu sia.

Caratteristiche
- Chat in streaming in tempo reale
- Selezione del modello
- Ricerca e gestione delle conversazioni
- Input vocale (speech-to-text)
- Caricamento di file e immagini per la generazione aumentata da recupero (RAG)
- Supporto per visione e multi-modale
- Rendering Markdown con evidenziazione della sintassi
- Temi chiaro, scuro e di sistema
- Archiviazione sicura delle credenziali (Keychain/Keystore)
- Esperienza ottimizzata per l'utilizzo offline

Requisiti
- Richiede un server Open-WebUI già esistente. Conduit non ospita né fornisce modelli AI.
- Per impostazione predefinita, nessun dato viene inviato a servizi di terze parti; tutto rimane sul tuo server configurato.

Permessi
- Microfono: Input vocale
- Fotocamera e Foto/Archiviazione: Allegati di immagini/file
- Rete: Connessione al tuo server Open-WebUI
