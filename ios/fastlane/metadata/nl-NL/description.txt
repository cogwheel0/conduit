Conduit is een open-source, native mobiele client voor Open‑WebUI. Verbind met je eigen server om te chatten met AI-modellen, gesprekken te beheren en je zelf gehoste AI veilig mee te nemen—onderweg en beveiligd.

Functies
- Real-time streaming chat
- Modelselectie
- Gesprekszoekfunctie en -beheer
- Spraakinvoer (spraak-naar-tekst)
- Bestand- en afbeeldinguploads voor retrieval-augmented generation (RAG)
- Spraakoproepen met on-device verwerking
- Toolondersteuning
- Markdown rendering met syntax highlighting
- Licht, donker en systeemthema's
- Veilige opslag van inloggegevens (Keychain/Keystore)

Vereisten
- Vereist een bestaande Open‑WebUI server. Conduit host geen AI-modellen en biedt deze niet aan.
- Standaard worden geen gegevens naar derde partijen verzonden; alles blijft bij je geconfigureerde server.

Machtigingen
- Microfoon: Spraakinvoer
- Camera en Foto's/Opslag: Afbeelding/bestandsbijlagen
- Netwerk: Verbinding met je Open‑WebUI server

Open Source:
Conduit is een open-source project. Voor ondersteuning, om problemen te melden of om de broncode te bekijken, bezoek onze GitHub repository:
