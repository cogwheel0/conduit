Conduit è un client mobile nativo open-source per Open‑WebUI. Connettiti al tuo server per chattare con modelli di IA, gestire conversazioni e portare con te la tua IA self-hosted—in modo sicuro e in movimento.

Funzionalità
- Chat in streaming in tempo reale
- Selezione modelli
- Ricerca e gestione conversazioni
- Input vocale (riconoscimento vocale)
- Caricamento file e immagini per generazione aumentata dal recupero (RAG)
- Chiamate vocali con elaborazione sul dispositivo
- Supporto strumenti
- Rendering Markdown con evidenziazione sintassi
- Temi chiaro, scuro e sistema
- Archiviazione sicura credenziali (Keychain/Keystore)

Requisiti
- Richiede un server Open‑WebUI esistente. Conduit non ospita né fornisce modelli di IA.
- Nessun dato viene inviato a servizi di terze parti per impostazione predefinita; tutto rimane sul tuo server configurato.

Autorizzazioni
- Microfono: Input vocale
- Fotocamera e Foto/Archiviazione: Allegati immagini/file
- Rete: Connessione al tuo server Open‑WebUI
