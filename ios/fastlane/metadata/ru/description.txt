Conduit — это клиент с открытым исходным кодом для Open‑WebUI. Подключитесь к своему серверу, чтобы общаться с моделями ИИ, управлять разговорами и брать свой самодельный ИИ с собой — безопасно и в пути.

Возможности
- Чат в реальном времени с потоковой передачей
- Выбор модели
- Поиск и управление разговорами
- Голосовой ввод (речь в текст)
- Загрузка файлов и изображений для генерации с расширением извлечения (RAG)
- Голосовые звонки с обработкой на устройстве
- Поддержка инструментов
- Рендеринг Markdown с подсветкой синтаксиса
- Светлая, тёмная и системная темы
- Безопасное хранилище учётных данных (Keychain/Keystore)

Требования
- Требуется существующий сервер Open‑WebUI. Conduit не размещает и не предоставляет модели ИИ.
- По умолчанию данные не отправляются сторонним службам; всё остаётся на вашем настроенном сервере.

Разрешения
- Микрофон: Голосовой ввод
- Камера и Фото/Хранилище: Вложения изображений/файлов
- Сеть: Подключение к вашему серверу Open‑WebUI
